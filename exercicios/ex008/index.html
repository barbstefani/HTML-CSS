Trabalhando com listas

Listas ordenadas

Para fazer uma lista ordenada use a tag <ol>, dentro dessa tag, use a tag <li> para enumerar cada item da lista, ele ordenara automaticamente independente de que local da lista voce adicionar o item novo. A tag </li> para fechamento é opcional

  1. acordar
  2. levantar da cama
  3. ir no banheiro
  4. tomar cafe

Listas não ordenadas

Para as listas não ordenadas apenas mude a tag <ol> para a tag <ul>

  • acordar
  • levantar da cama
  • ir no banheiro
  • tomar cafe
